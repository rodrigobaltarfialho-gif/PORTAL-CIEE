# Fotos dos funcionários

Coloque as fotos nesta pasta em formato `.jpg`, `.jpeg` ou `.png`.

O nome do arquivo deve seguir o nome do funcionário sem acentos, em minúsculas e com hífen.

Exemplos:

- `Ana Silva` -> `ana-silva.jpg`
- `João Souza (estagiário)` -> `joao-souza.jpg`
- `Maria de Oliveira` -> `maria-de-oliveira.jpg`

Se a foto não existir, o portal mostra automaticamente um avatar com as iniciais.

O portal também tenta carregar arquivos `.png.png` caso o Windows duplique a extensão sem querer.
